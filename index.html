🎯 Kế hoạch năm
📖 Hướng dẫn cai dat Supabase

Bước 1 — Vao supabase.com → Sign Up → New Project

Bước 2 — SQL Editor → paste va chay doan SQL sau:

-- Bảng số liệu thực hiện create table if not exists bv_data ( id bigserial primary key, year int not null, month int not null, dept text not null, indicator text not null, thuc_hien numeric default 0, ck numeric default 0, updated_at timestamptz default now(), unique(year, month, dept, indicator) ); -- Bảng kế hoạch năm create table if not exists bv_kh ( id bigserial primary key, year int not null, dept text not null, indicator text not null, kh_nam numeric default 0, updated_at timestamptz default now(), unique(year, dept, indicator) ); -- Bảng mật khẩu khoa create table if not exists bv_passwords ( dept text primary key, password text not null default '1' ); -- Policies alter table bv_data enable row level security; alter table bv_kh enable row level security; alter table bv_passwords enable row level security; create policy pub_read_data on bv_data for select using (true); create policy pub_read_kh on bv_kh for select using (true); create policy pub_read_pw on bv_passwords for select using (true); create policy pub_ins_data on bv_data for insert with check (true); create policy pub_ins_kh on bv_kh for insert with check (true); create policy pub_ins_pw on bv_passwords for insert with check (true); create policy pub_upd_data on bv_data for update using (true); create policy pub_upd_kh on bv_kh for update using (true); create policy pub_upd_pw on bv_passwords for update using (true);

Bước 3 — Project Settings → API → copy Project URL va anon public key → dan vao man hinh Setup

Bước 4 — Upload file HTML lên GitHub → Deploy Vercel → xong!

✅ Sau khi xong: Nhap KH năm truoc → Nhập số thực hiện hang thang → Dashboard tự hiển thị.
